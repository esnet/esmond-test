add_to_payload(item)[source]¶

Adds a new dict element to the object payload.

send_data()[source]¶

Format current payload, send to REST api and clear the payload. Payload is cleared after a write so as not to send duplicate data, and so the same instance can be used to send multiple times.

set_payload(payload)[source]¶

Sets object payload to a complete list of dicts passed in. This will overwrite the internal payload if any elements had been previously defined (and issue a warning).
